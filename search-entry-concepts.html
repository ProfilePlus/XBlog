Search Entry Prototype Set

这三套只讨论首页怎么“接入口”，不讨论搜索结果页本身。

三套都默认搜索是独立 `/search` 页面，区别只在于首页入口放在哪、曝光度多高、对现有布局影响多大。

A

头部右侧胶囊

搜索只出现在头部右侧 action 区，中间导航和 Hero 都不动。

  • 影响最小
  • 最稳
  • 我最推荐先用这套
打开 A
B

Hero 次 CTA

把 Hero 的次按钮改成“搜索文章”，入口曝光度最高。

  • 最容易被看到
  • 会改变首屏叙事重心
  • 适合想强推搜索功能
打开 B
C

头部下方引导条

在头部和 Hero 之间加一条轻搜索引导层，曝光度介于 A 和 B 之间。

  • 折中方案
  • 新功能提示感更强
  • 会略微增加首屏高度
打开 C
